# Launch Blockers

This document tracks critical issues that prevent successful completion of core user flows.

## Priority Definitions

- **CRITICAL**: Blocks core functionality, causes data loss, or prevents successful completion of essential flows
- **HIGH**: Significant impact on user experience, but doesn't completely block core functionality
